# 紫微斗数 · 精成

基于《紫微斗数精成》（大德山人 编著）构建的**专业紫微斗数排盘与解读系统**。

- ✨ 纯前端、**单页 HTML**，离线即用
- 🔮 标准紫微斗数排盘算法（农历转换 / 十四主星 / 辅星煞星 / 四化 / 大限流年）
- 🎨 传统 **12 宫方盘** SVG 可视化
- 📖 基于 PDF 原文构建的规则 + 知识库，逐宫专业解读
- 💬 可针对事业 / 财运 / 姻缘 / 健康 等宫位**追问深挖**

## 快速开始

因为使用了 ES Module，需要通过本地静态服务器打开：

```bash
cd ziwei-system
python3 -m http.server 8765
# 浏览器访问 http://localhost:8765/
```

## 重建知识库

如需重新抽取 PDF：

```bash
pip3 install pymupdf
python3 scripts/extract_pdf.py   # 解析 PDF → data/raw_text.json
python3 scripts/build_kb.py      # 产出 src/kb/*.js
```

## 测试

```bash
node tests/regression.mjs
```

## 功能模块

### 排盘输入

- 公历日期（1900-01-31 ~ 2100-12-31）
- 出生时辰（12 时辰）
- 性别（影响大限顺逆行）

### 排盘产出

- 农历日期、四柱八字
- 五行局（水二/木三/金四/土五/火六）、纳音
- 命主、身主、命宫、身宫
- 十二宫完整星曜布局（主星 + 辅星 + 煞星）
- 星曜庙旺陷
- 生年四化（禄/权/科/忌）
- 大限 12 段、小限、流年

### 解读模块

1. **命宫总论**：主星性情 + 身宫关系
2. **格局判断**：紫府朝垣 / 府相朝垣 / 君臣庆会 / 机月同梁 / 杀破狼 / 火贪 / 铃贪 / 禄马交驰 / 日月并明 / 日月反背 / 孤君寡人 等
3. **十二宫逐宫详解**：每宫宫意 + 主星 + 组合 + 辅煞 + 四化应验
4. **四化飞星**：化禄/权/科/忌入宫解读
5. **大限流年**：当前大限主事 + 本年流年重点
6. **追问**：事业/财运/姻缘/健康/子女/父母/兄弟/朋友/田宅/福德/迁移 等
7. **🗣 大白话翻译（总开关）**：页面顶部切换开关，一键把古文/术语风格解读转换成现代口语
   - 150+ 条术语词典（星曜/宫位/四化/格局/运限/文言助词）
   - 保护专业名词（财帛主、将星、命主、官禄主等）不被误伤
   - 句式改写（"紫微入命" → "你命宫里有紫微"）
8. **🕰 流年历史回顾**：自动排出过去 5/8/10/15 年的流年命盘
   - 每年给出四维度解读（事业/财运/感情/健康）+ 总评（大吉/吉顺/平顺偏吉/平中带险/多变动/需谨慎）
   - 每年旁边三档打分按钮：✅ 很准 / 🤔 一般 / ❌ 不准
   - 自动统计命中率（很准=100% / 一般=50% / 不准=0%）并出具准确率报告
   - 打分数据自动保存到浏览器 localStorage（按命盘独立存储）
   - 支持 JSON 导出/导入，方便长期备份与跨设备同步

## 算法说明

### 安星诀

- **紫微**：依五行局数与农历日查表（与《紫微斗数全书》一致）
- **十四主星**：紫微系（紫→机→阳→武→同→廉）逆行 + 天府系（府→阴→贪→巨→相→梁→杀→破）顺行
- **四化**："钦天四化"流派（主流版本）
- **起大限**：阳男阴女顺行、阴男阳女逆行；水二/木三/金四/土五/火六起运岁数

### 农历转换

基于 1900-2100 年 `LUNAR_INFO` 压缩表（业界通用），与香港天文台输出一致。

## 回归测试验证点

覆盖 5 个历史日期、20 个验证点，包括：
- 农历转换（跨年 / 闰月）
- 年柱 / 日柱 / 五行局
- 紫微位置 / 十四主星落宫
- 四化判定 / 化星落宫
- 大限起运岁数
